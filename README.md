# Personal Expense Tracker (Socket-Based)

This project is a personal expense tracking application built using:

- Python TCP Socket Programming (Backend)
- MySQL Database
- Flutter Mobile Application (Frontend)

## Key Features
- User authentication (Login / Signup)
- Add, view, and delete expenses
- Expense categorization
- Network-based communication using raw TCP sockets

## Project Structure
- `backend/` – Python socket server and database logic
- `frontend/` – Flutter mobile application

This project is developed as part of an academic assignment to demonstrate
socket programming integrated with a database and a mobile client.
